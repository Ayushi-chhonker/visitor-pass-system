# Visitor Pass Management System

 Project Overview

Visitor Pass Management System is a full-stack web application used to manage visitors, appointments, and digital visitor passes with QR code generation.

The system allows:
- Visitor registration
- Appointment creation
- Appointment approval
- QR pass generation
- Visitor check-in/check-out
- Dashboard monitoring

--

 Technologies Used

1 Frontend
- React.js
- Axios
- React Router DOM

2 Backend
- Node.js
- Express.js

- MongoDB Atlas

- QRCode Generator
